Markdown语法摘要

发表于 2018-07-05 | 更新于: 2018-07-05 | 分类于 工具
字数统计: 222 字

基本语法

  1. 换行 两个空格 + enter
  2. 分段 两个enter
  3. 空格 shift+space切换为全角模式　+　space
  4. 标题 #~######,标题级数与#个数一致,若无效考虑后接一个空格
  5. 强调 **文字**
  6. 转意 \
  7. 引用 >
  8. 表格 |分割行,-分割列,:控制对齐方式
  9. 链接 [文字描述](url)
  10. 图片 ![图片描述](url)
  11. 格式 <font face="微软雅黑" size=5 color=#0099ff>微软雅黑字体</font>
  12. 列表 * + - 1.任选其一
